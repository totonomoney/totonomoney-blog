こんにちは、まゆみんです。

シングルマザーになってから、一番プレッシャーを感じたのは「教育費」でした。

「いくらかかるの？」「足りるの？」

子どもが小さいうちから、頭の片隅にずっとある不安です。

この記事では、私が実際に経験した高校〜大学の教育費を全部公開します。

正直に書くので、参考にしてもらえると嬉しいです。

---

## 結論：2人分の教育費は想像以上だった

先に結論から言います。

**受験から入学までの費用だけで、2人合わせると200万円以上かかりました。**

これは「受験料・塾代・入学金・引越し費用」など受験〜大学1年目前半までの合計です。

毎月の授業料や仕送りは含んでいません。

最初に知っておきたかった、というのが正直な気持ちです。

---

## 高校受験にかかった費用【次女の場合】

次女は**塾なしで高校受験**に挑みました。

本人が「学校のワークだけでやってみる」と言って、それを信じました。

受験したのは**私立1校・県立1校の計2校**。

結果として県立の志望校に合格できたのですが、これは次女が努力してくれたおかげです。

かかった費用はこの通りです。

| 項目 | 金額 |
|------|------|
| 受験料（私立1校） | 約22,000円 |
| 受験料（県立1校） | 2,200円 |
| 模試代（年間） | 約1〜2万円 |
| 入学金・制服・教材・PTA費など（県立） | 約20〜25万円 |

受験料は2校合わせても**約24,000円**と、大学受験に比べればずっとリーズナブルです。

でも入学時にまとめてかかる制服・教材・PTA費などが、シンママには正直きつかった。

しかも入学後すぐに、次女本人から「先取り勉強したい」と塾を希望されたので、そこからさらに毎月の出費が始まりました。

塾なしで乗り切れたことで受験までは助かりましたが、**入学後の費用は別で考えておく必要があります。**

---

## 大学受験にかかった費用【長女の場合】

長女は**県外の私立大学**を第一志望にしていました。

「学費の安い地元の国公立の方が…」と思わなかったと言えば嘘になります。

でも、娘が自分で決めた道を、親の都合で止める気にはなれなかった。

「費用は気にせず、自分が後悔しない大学を選んでいいよ」と言いました。

内心は不安でいっぱいでした。

実際にかかった受験費用はこちらです。

| 項目 | 金額 |
|------|------|
| 模試代（年間） | 約3〜5万円 |
| 塾代（月） | 約4〜7万円 |
| 受験料（複数校） | 1校あたり約3.5万円 |
| 交通費・宿泊費（遠征） | 約2〜3万円 |

塾代だけで年間50〜80万円規模になります。

受験費用の合計は、**長女だけで年間80〜100万円**ほどになりました。

---

## 大学入学時にかかった費用

長女が合格したとき、泣きました。

離婚してぐちゃぐちゃのまま走り続けた一年を、一緒に乗り越えられたんだという気持ちがこみ上げてきました。

少し落ち着いてから、通帳を開きました。

「…思ってたより、ない」

入学時にかかった費用がこちらです。

| 項目 | 金額 |
|------|------|
| 入学金 | 約20万円 |
| 前期授業料・施設費等 | 約50万円 |
| 敷金・保証料など（一人暮らし） | 約12万円 |
| 家賃（初月） | 約6万円 |
| 火災保険・清掃料など | 約5万円 |
| 引越し費用（自力対応） | 交通費・ガソリン代のみ |

**入学時だけで軽く100万円を超えました。**

シングルになってから奨学金の申請が間に合わず、入学後に慌てて手続きをしたのも正直なところです。

---

## 毎月の仕送りと授業料

入学後も出費は続きます。

| 項目 | 金額 |
|------|------|
| 仕送り | 月5〜6万円 |
| 授業料（後期以降） | 前期と同額 |
| 奨学金（給付型・貸与型） | 一部補てん |

## 固定費の見直しで乗り切った

お金が出ていく一方で、私がやったことを正直に書きます。

### スマホ代を見直した（年間12万円削減）

3人分（私・長女・次女）を大手キャリアから格安SIMに切り替えました。

それだけで**3人合計で月1万円以下**になりました。

以前は3人で月2万円以上払っていたので、年間約12万円の削減です。

「面倒そう」と思って後回しにしていましたが、手続きは1日で終わりました。

詳しくはこちら→ [スマホ代を月5,000円下げた方法（格安SIM比較）](/posts/sim-hikaku/)

### ひとり親支援制度を全部申請した

市役所に相談して、受けられる制度をすべて申請しました。

「どうせ対象外だろう」と思っていたものの、実際に申請してみると複数の支援を受けられました。

申請しないともらえない制度がほとんどです。まず電話してみるだけでいいです。

詳しくはこちら→ [ひとり親家庭がもらえるお金・制度まとめ](/posts/hitorioya-seido/)

### 養育費をしっかり確保した

離婚後の生活を安定させるために、養育費の交渉に弁護士を入れました。

1人10万円×2人＝月20万円、大学卒業まで確保できました。

養育費の確保が、その後の生活の安定に直結しました。

詳しくはこちら→ [協議離婚でも養育費月20万円もらえた話](/posts/youikuhi/)

## 教育費の準備で後悔していること

正直に言うと、**もっと早くNISAを本格化しておけばよかった**と思っています。

離婚前から少し始めていましたが、本腰を入れたのは離婚後でした。

今は月10万円（つみたて投資枠の上限）を積み立てています。

「年金だけでは不安」から「自分で計算できるようになった」に変わったのは、FP2級を取得してからです。

## まとめ

教育費の目安（2人分・受験〜入学まで）

| 内容 | 金額目安 |
|------|---------|
| 高校受験〜入学（2校受験・県立合格） | 約23〜29万円 |
| 大学受験費用（塾含む） | 約80〜100万円 |
| 大学入学時費用 | 約100万円以上 |
| **合計（受験〜大学入学まで）** | **約200万円以上** |

毎月の仕送り・授業料はこれとは別にかかります。

「想像よりかかる」というのが正直なところです。

でも固定費を見直すだけで、毎月の支出は確実に減らせます。

まず一番効果が大きいのは**スマホ代の見直し**です。手続きは1日で終わります。

同じように教育費で悩んでいる方の、少しでも参考になれば嬉しいです 🌿
